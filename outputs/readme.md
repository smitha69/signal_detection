# ATS / HR Tech Signal Detector

A signal detection system that identifies companies upgrading their 
HR and recruitment technology stack (ATS modernization signals).

Built for the Vikaas.ai Signal Detection Assignment — Task 4.

---

## What It Does

- Fetches articles from public HR/Tech RSS news feeds
- Searches for ATS and HR tech keywords (Workday, SuccessFactors, etc.)
- Scores each signal from 0 to 100
- Detects modernization stage: planning / implementing / completed
- Saves results as structured JSON
